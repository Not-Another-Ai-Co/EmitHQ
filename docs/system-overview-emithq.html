Design Principles

  • Persist before enqueue — messages saved to PostgreSQL before Redis queue. Queue failure = recoverable.
  • Standard Webhooks — HMAC-SHA256 signing compatible with the industry spec.
  • Row-Level Security — PostgreSQL RLS enforces tenant isolation at the database layer.
  • Edge for speed, origin for reliability — Cloudflare Workers ack in <50ms, Railway workers deliver.
DEC-004: Hybrid edge/origin architecture. CF Workers for fast inbound, Railway for persistent queue workers and DB access.
